# AWS Certified Generative AI - Specialty (ANT-C01)

Study notes for the AWS Generative AI Professional certification.

## Exam Domains

| Domain | Weight | Topics |
|--------|--------|--------|
| 1. Fundamentals of AI and ML | 14% | AI/ML concepts, generative AI fundamentals |
| 2. Fundamentals of Generative AI | 22% | Foundation models, training, fine-tuning, prompt engineering |
| 3. Applications of Foundation Models | 28% | RAG, agents, model selection, responsible AI |
| 4. Guidelines for Responsible AI | 14% | Bias, fairness, transparency, governance |
| 5. Security, Compliance, and Governance | 22% | Data security, IAM, model security, compliance |
